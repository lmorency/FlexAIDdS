FlexAID∆S

Entropy-Driven Molecular Docking Engine

Genetic algorithms meet statistical mechanics for real-world drug discovery

0.93 Pearson r (ITC-187)
1.4 RMSE kcal/mol
92% Correct binding mode
Get Started View on GitHub
Most docking engines optimize enthalpy alone. FlexAID∆S integrates Shannon entropy into its scoring via a full statistical mechanics framework — recovering the correct binding mode 92% of the time when enthalpy-only scoring fails.

Capabilities

Docking Engine

  • Genetic algorithm with configurable population, crossover, mutation, selection
  • Voronoi contact function (CF) for shape complementarity scoring
  • Dead-end elimination (DEE) torsion pruning
  • Zero-copy batch scoring via VoronoiCFBatch with OpenMP

Thermodynamics

  • Statistical mechanics engine: partition function, free energy, heat capacity, entropy
  • Shannon entropy scoring + torsional vibrational entropy
  • Torsional ENCoM (tENCoM) backbone flexibility

Molecular Flexibility

  • Full flexibility — ligand torsions, ring conformers, chirality
  • Non-aromatic ring conformer sampling and sugar pucker
  • Explicit R/S stereocenter discrimination
  • PDB, CIF/mmCIF, MOL2, SDF/MOL, SMILES input with auto 3D generation
  • Idiotproof CLI — auto-detects receptor vs. ligand, any argument order

Hardware Acceleration

  • Unified hardware dispatch — CUDA > ROCm/HIP > Metal > AVX-512 > AVX2 > OpenMP > scalar
  • ROCm/HIP support for AMD Instinct MI100, MI200, MI300 GPUs
  • SIMD primitives: AVX-512 and AVX2 vectorized geometric computations
  • Ultra-fast HPC binaries with LTO + -march=native
  • TurboQuant vector quantization — near-optimal compression of 256-dim contact vectors

Bonhomme Fleet

  • Distributed docking across Apple devices via iCloud Drive
  • Device-aware scheduling — battery, thermal state, TFLOPS weighting
  • ChaChaPoly-encrypted work chunks for secure computation
  • Intelligence oracle — on-device AI via Apple FoundationModels
  • TypeScript PWA — real-time Fleet dashboard + Mol* 3D viewer

Analysis & Integration

  • Python package flexaidds — API, CLI, PyMOL plugin
  • ProcessLigand — pure C++20 SMILES parser, 3D builder, atom typing (no RDKit/Boost)
  • NATURaL co-translational assembly
  • 50,000+ LOC codebase — FastOPTICS + Density Peak clustering

Usage

bash
# Full flexibility dock — entropy at 300K
./FlexAIDdS receptor.pdb ligand.mol2

# Argument order doesn't matter — auto-detected from content
./FlexAIDdS ligand.mol2 receptor.pdb

# Dock from SMILES — 3D coordinates built automatically
./FlexAIDdS receptor.pdb "CC(=O)Oc1ccccc1C(=O)O"

# CIF/mmCIF receptor input
./FlexAIDdS receptor.cif ligand.sdf

# Override parameters via JSON config
./FlexAIDdS receptor.pdb ligand.mol2 -c config.json

# Fast rigid screening
./FlexAIDdS receptor.pdb ligand.mol2 --rigid
python
import flexaidds as fd

# High-level docking
results = fd.dock(
    receptor='receptor.pdb',
    ligand='ligand.mol2',
    binding_site='auto',
    compute_entropy=True,
)

# Load and analyze existing results
run = fd.load_results("path/to/output_dir")
print(run.binding_modes[0].free_energy)

# Thermodynamic analysis
engine = fd.StatMechEngine(temperature=300.0)
engine.add_sample(-7.5)
thermo = engine.compute()
print("F =", thermo.free_energy)
print("S =", thermo.entropy)

# Phase 3 — Vibrational entropy correction
from flexaidds.tencm import TorsionalENM, run_shannon_thermo_stack

tenm = TorsionalENM("apo.pdb", "holo.pdb")
delta_s_vib = tenm.compute_delta_s(temperature=300.0)

# Shannon configurational entropy (GPU-accelerated)
S_config = run_shannon_thermo_stack(dihedral_angles, n_bins=36)

bash
# Compute vibrational entropy differential between structures
tENCoM reference.pdb target1.pdb [target2.pdb ...]

# Custom temperature and cutoff
tENCoM reference.pdb mutant.pdb -T 310 -r 9.0

# Specify spring constant and output prefix
tENCoM apo.pdb holo.pdb -k 1.0 -o my_analysis

# Batch analysis of multiple conformers
tENCoM wildtype.pdb conf1.pdb conf2.pdb conf3.pdb -T 300

tENCoM computes torsional normal modes via Jacobi diagonalisation of Cα elastic network contacts, reporting ΔSvib and B-factor differentials between reference and target structures.

bash
# Auto-detected: nucleotide ligand triggers co-translational mode
./FlexAIDdS ribosome.pdb atp_analog.mol2

# RNA polymerase + nascent RNA — co-transcriptional mode
./FlexAIDdS rnap_complex.pdb rna_fragment.mol2

# Skip chain growth — treat receptor as fully folded
./FlexAIDdS ribosome.pdb atp_analog.mol2 --folded

NATURaL activates automatically for nucleotide systems. It grows the receptor chain residue-by-residue at ribosome speed (Zhao 2011), identifies pause sites, computes incremental CF + Shannon entropy, and models TM helix insertion via Sec61 translocon (Hessa 2007). Supported organisms: E. coli K-12 and Human HEK293.
