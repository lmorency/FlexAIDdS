# CASF-2016 Benchmark

The Comparative Assessment of Scoring Functions (CASF-2016) evaluates Scoring Power, Ranking Power, Docking Power, and Screening Power on 285 high-quality protein-ligand complexes.

## Running

```bash
export CASF2016_DATA=/path/to/PDBbind/CASF-2016
python tests/benchmarks/casf2016/run_casf2016.py --results /path/to/results
```

## Metrics

- **Scoring Power**: Pearson R between predicted and experimental pKd
- **Docking Power**: Success rate at 1Å/2Å/3Å RMSD thresholds
